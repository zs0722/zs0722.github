学习心得

固定格式

为固定格式，到学习函数时才改变 main 的函数名称

变量

定义

输入的数据放到变量中， int m = 0 ，其中 ‘int’为 m 的 类型， m 为变量名 意为名叫m的int变量

一般形式为

<类型名称> <变量名称>;

<类型名称> <变量名称> ,<变量名称>;

**注意分号 逗号

变量的名

变量名即为标识符，（后面可能用来表示函数）可为字母，数字，下划线，数字不能出现在第一位，c语言关键字不能为标识符。

赋值和初始化

int m = 0 “=” 为赋值运算符，把右边的给左边，因为是在定义变量开始赋值，所以也叫初始化 初始值是 0 .

int m = 0，h = 100；

赋值 a = b ， 与 b = a ， 意义不同 b 给 a ， a 给 b 。

** 定义一个变量一定要初始化

变量类型

所有变量使用前必须声明类型

数据输入

scanf (“%d”, &m ) scanf函数读入下一个整数，读到的结果给 m ，注意 &（学习指针时说明）
